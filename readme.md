### 欢迎使用FireBird StockAnalysis
FireBird StockAnalysis从网络股票行情服务器中提取实时行情和历史数据，将处理后的信息存储于数据库中。

#### 功能
1.分别从新浪、腾讯和网易实时行情服务器中提取实时交易数据，保证读取到所有的Level 1数据。

2.从Finnhub.io、Tiingo.com中收取美国股票市场的各种数据。

3.分析提取的实时数据其具体交易情况。

4.从网易历史数据服务器中提取日线历史数据。
